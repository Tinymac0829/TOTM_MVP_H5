## 启动链路

- [ ] 打开页面后默认显示开始界面
- [ ] 点击“开始游戏”后进入加载态
- [ ] 加载完成后进入 `story_001`
- [ ] 进入玩法后角色可以正常响应输入

## 移动与输入

- [ ] 角色四向滑行直到受阻才停
- [ ] 触屏或键盘输入响应正常
- [ ] HUD 顶部栏显示期间不影响正常游玩
- [ ] 弹窗显示期间继续滑动不会触发移动

## 收集与 HUD

- [ ] 经过 Dot 时地图上的 Dot 立即消失
- [ ] 经过 Coin 时地图上的 Coin 立即消失
- [ ] 经过 Star 时地图上的 Star 立即消失
- [ ] HUD 顶部 `星 / 币 / 点` 计数实时更新
- [ ] HUD 顶部栏始终位于最上层
- [ ] `1080x1920` 下 HUD 不遮挡核心可玩区

## 失败链路

- [ ] 已通过 `?stage=eng04_death_validation` 进入本地验证关卡
- [ ] 触发死亡后进入失败弹窗
- [ ] 失败弹窗淡入期间按钮不可点击
- [ ] 失败弹窗显示期间玩法 fixedUpdate 已暂停
- [ ] 点击“重新开始”后回到当前关卡 Enter
- [ ] 重开后收集物恢复
- [ ] 重开后 HUD 计数归零
- [ ] 失败后可在 `2s` 需求口径内重开

## 通关闭环

- [ ] 到达 Exit 后先完成进入出口格子的动作
- [ ] 之后进入通关弹窗
- [ ] 通关弹窗显示最终 `星 / 币 / 点` 统计
- [ ] 通关弹窗淡入期间按钮不可点击
- [ ] 当前仅接入 `story_001` 时，按钮行为符合重复游玩 fallback 口径
- [ ] 点击通关按钮后重新进入 `story_001`
- [ ] 重进后关卡可再次正常游玩

## 点击与缩放

- [ ] 按钮外区域点击不会误触发
- [ ] 按钮点击区域与视觉位置一致
- [ ] 改变浏览器窗口尺寸后点击区域仍正确
- [ ] 不同设备或窗口比例下没有明显点击偏移

## 异常与恢复

- [ ] 若加载失败，界面会回到可恢复状态
- [ ] 错误提示可见且可理解
- [ ] 不会出现卡死在不可交互状态的问题
